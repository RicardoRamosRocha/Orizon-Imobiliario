# Consultoria Jurídica

## Objetivo

Centralizar todo o atendimento jurídico relacionado às negociações imobiliárias, permitindo acompanhar consultas, orientações, documentos e pareceres.

---

# Funcionalidades

* Cadastro de consultas jurídicas
* Atendimento presencial ou online
* Histórico completo do cliente
* Associação com imóveis
* Associação com contratos
* Upload de documentos
* Emissão de pareceres
* Registro de observações
* Controle de prazos
* Status do atendimento

---

# Fluxo

Cliente → Solicitação → Análise Inicial → Documentação → Parecer Jurídico → Encerramento

---

# Dados Principais

* Cliente
* Imóvel
* Responsável
* Data
* Tipo da consulta
* Descrição
* Situação
* Prioridade
* Documentos anexos

---

# Benefícios

* Centralização dos atendimentos
* Histórico jurídico completo
* Maior segurança nas negociações
* Organização dos processos
