# Meus Imóveis

## Objetivo

Permitir que o cliente visualize todos os imóveis vinculados ao seu cadastro.

## Informações exibidas

* Foto principal
* Código do imóvel
* Tipo
* Endereço
* Situação
* Valor
* Corretor responsável
* Data de atualização

## Funcionalidades

* Visualizar detalhes
* Acompanhar negociação
* Baixar documentos
* Agendar visita
* Compartilhar imóvel
* Favoritar imóveis

## Filtros

* Compra
* Venda
* Locação
* Ativos
* Finalizados

## Status

* Em negociação
* Proposta enviada
* Em análise
* Documentação
* Contrato
* Concluído
