# Perfil

## Objetivo

Permitir que o cliente gerencie seus dados cadastrais, preferências e configurações de acesso.

## Dados Cadastrais

* Nome completo
* CPF/CNPJ
* Data de nascimento
* Estado civil
* Endereço
* Telefones
* E-mail

## Configurações

* Alteração de senha
* Autenticação em dois fatores (2FA)
* Idioma
* Tema claro/escuro
* Preferências de notificação

## Funcionalidades

* Atualizar cadastro
* Alterar foto de perfil
* Gerenciar dispositivos conectados
* Consultar histórico de acessos
* Solicitar exclusão da conta (LGPD)

## Segurança

* Criptografia de dados
* Confirmação por e-mail
* Registro de acessos
* Controle de sessões ativas
