# Dashboard do Cliente

## Objetivo

Apresentar ao cliente todas as informações relevantes em uma única tela.

## Componentes

* Boas-vindas
* Resumo dos imóveis
* Propostas em andamento
* Pendências documentais
* Próximas visitas
* Parcelas a vencer
* Últimas mensagens
* Notificações

## Indicadores

* Imóveis ativos
* Contratos vigentes
* Documentos pendentes
* Pagamentos pendentes
* Solicitações abertas

## Ações rápidas

* Ver imóveis
* Enviar documentos
* Abrir atendimento
* Atualizar cadastro
* Consultar financeiro
