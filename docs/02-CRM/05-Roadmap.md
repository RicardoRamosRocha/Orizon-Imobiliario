# Critérios de Conclusão

A Sprint será considerada concluída quando:

* Todos os módulos estiverem implementados.
* O fluxo Lead → Cliente estiver funcionando integralmente.
* O funil comercial permitir acompanhar todas as oportunidades.
* Atendimentos, tarefas e agenda estiverem integrados ao CRM.
* O dashboard apresentar indicadores atualizados em tempo real.
* A interface seguir o Design System padrão do Orizon Imobiliário.
